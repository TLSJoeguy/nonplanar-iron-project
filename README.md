# NONPLANAR IRONING  
### NOW USER FRIENDLY!  
This project is currently a work in progress, but it works good enough, so I though I'd publish it as is for the time being.  
This project aims to implement high-fidelity, general nonplanar ironing for FDM 3D printing. It is designed to work with PrusaSlicer, though ultimately it can be easily adapted to use other slicers’ 3MF formats.
